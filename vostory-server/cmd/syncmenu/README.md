# 菜单同步工具 (Menu Synchronization Tool)

该工具用于扫描 HTTP API 路由，并将其同步到系统菜单表中。

## 功能

- 扫描 `internal/server/http.go` 中定义的所有 API 路由
- 按层级关系构建菜单树
- 与数据库中的菜单数据进行同步：
  - 新增不存在的菜单
  - 更新已存在的菜单
  - 软删除不再需要的菜单（如果没有子菜单）

## 使用方法

```bash
# 使用默认配置文件路径
./syncmenu

# 指定配置文件路径
./syncmenu -conf ./config/prod.yml
```

## 工作原理

1. 工具解析 `http.go` 文件中的路由定义，提取路由组和具体路由
2. 构建菜单树结构，保持层级关系
3. 根据菜单树，与数据库中的菜单表进行比对
4. 更新数据库中的菜单，保持与代码中的路由定义一致

## 菜单类型映射

- 路由组 → 目录类型菜单 (M)
- 无子项的路由组 → 菜单类型 (C)
- 具体路由 → 功能/按钮类型 (F)

## 注意事项

- 运行前请确保配置文件中的数据库连接信息正确
- 该工具默认不会删除有子菜单的菜单项，即使其不再对应任何路由
- 生成的菜单名称基于处理器名称，可能需要手动调整以提高可读性
